# Offline Release Manager

A lightweight, offline-first launcher-based system for managing
versioned application releases in constrained environments.

## Key Concepts
- Immutable version storage
- Manifest-driven version selection
- Safe rollback without reinstall
- Deterministic execution
- Offline-only operation

## Use Cases
- Air-gapped systems
- Regulated environments
- Industrial or embedded-adjacent software
